# Tunes-of-Randomness

## Random Music Generator

Welcome to the Tunes of Randomness! This is a simple music player application that I 
built in Visual Studio. 
Due to the size of the music files, they are hosted on OneDrive.

## How to Set Up the Project

### 1. Clone the Repository
To set up the project on your local device, follow these steps:

#### Step 1: Install Git (if not already installed)
- Download and install Git from [git-scm.com](https://git-scm.com/).
- Follow the installation instructions for your operating system.

#### Step 2: Open a Terminal
- **Windows:** Open Command Prompt or PowerShell.
- **Mac/Linux:** Open the Terminal app.

#### Step 3: Navigate to Your Desired Folder
Choose the folder where you want to store the project. Use the `cd` command to navigate to it:
```bash
cd path/to/your/folder
```

#### Step 4: Clone the Repository
Run the following command to download the project files:
```bash
git clone https://github.com/TinySpaceGremlin23/Tunes-of-Randomness.git
```

#### Step 5: Navigate to the Project Folder
After cloning, move into the project folder:
```bash
cd Tunes-of-Randomness
```
