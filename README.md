# Tunes-of-Randomness

Welcome to the **Tunes of Randomness!**  
This project is a simple music player web app  
that I made using basic coding tools (HTML, CSS, and JS).  

This is a personal project. I spent days—and even weeks—gathering songs  
from a saved playlist I had on YouTube. I downloaded them,  
converted them to `.mp3` format, and added them  
to a folder on a separate hosting service (since it's too big to host here).



# How to Set Up and Run the Project:

Download the Project Files:

-Go to the GitHub repository for this project.

-Click on the green "Code" button and select "Download ZIP."

-Extract the ZIP file to a folder on your computer.

# Download the Music Files:

-[Click here to download the music folder.](https://mega.nz/folder/HAhgwa6S#ucbOtqE40kib1YVhfo1zoA)  
-After downloading, extract the `music.zip` file. This will create a folder called `music`.

# Place the Music Folder in the Project:
-Move the extracted music folder into the project folder 

(the same folder where index.html is located).

# Open the Project in Your Browser:
-Double-click on the `index.html` file in the project folder  
to open it in your default web browser.  

That's it! The music player should now load the songs  
from the music folder and be ready to use.  

If you encounter any issues or have a song suggestion,  
click on the watermark in the corner. That leads to my  
Instagram, and you can DM me there!
